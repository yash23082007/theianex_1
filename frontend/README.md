task 1==Personal Portfolio Website
Build a full-stack personal portfolio to showcase your projects and skills.

Key Features:
Frontend: HTML, CSS, JavaScript (or React.js)
Backend: Node.js/Express.js or Django/Flask
Database: Store project details using MySQL, MongoDB, or PostgreSQL
Deploy website on a hosting platform (Heroku, Netlify, or Vercel)


## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

## Environment

Use `.env.example` as a starting point:

```bash
VITE_API_URL=http://localhost:5000
```

When `VITE_API_URL` is empty in development, Vite proxies `/api` to `http://localhost:5000`.
